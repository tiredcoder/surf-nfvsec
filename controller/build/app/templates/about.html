SURF NFV Controller v1.0

Author: Rik Janssen Project: UvA, SNE/OS3, RP2

  • This controller can be used to connect NFVs with educational institutions.
  • Manual/higher-level orchestration is required; this controller will not auto. create/delete rules.
  • Load balancing and function chaining are not implemented.
  • Security (authentication and encryption) and error checking are also not implemented.
